##👋 Hello, I'm Dr. Tom O'Shea |

I'm an Environmental Intelligence (EI) Research Fellow and Senior Research Associate working at the intersection of spatial modelling, behavioural analysis and decision‑support research.
I aim to build transparent, reproducible modelling workflows that help organisations understand how people interact with their environments.

## 🔍 What I work on:

- Spatial and agent‑based modelling  
- Gravity frameworks for greenspace visitation  
- Behavioural modelling and decision‑support tools  
- Reproducible scientific workflows  
- Visual storytelling through maps and figures  

## 🌱 Current focus:

I’m developing **ViSAGE**, a flexible modelling framework for simulating greenspace visitation patterns and testing the impacts of new developments. 
// Recent milestones for this framework include: multiple scenario modelling for the impact of different scales of development on the regional greenspace network and extensions for quality‑based attractiveness. //

## 🧠 Theoretical interests:

- Socio‑Spatial Perspective (SSP)  
- Actor‑Network Theory (ANT)  
- Applied Assemblage Theory (AAT)  
- Object‑Oriented Ontology (OOO)  

These approaches shape how I think about environments, behaviour and the relational structures that underpin spatial systems.

## 🧰 Skills and interests:

- Rigorous spatial modelling and simulation  
- Transparent, modular code design  
- Scientific communication and figure sequencing  
- Environmental intelligence for everyday lives  

## 📫 Get in touch:

I'm interested in discussing/developing different approaches for modelling, greenspace accessibility or reproducible workflows.
